## üìö KMP Architecture Lessons Learned

### ‚úÖ Proper KMP Patterns in commonMain:

**1. Use expect/actual for Platform-Specific Code:**
```kotlin
// In commonMain
expect object CryptoUtils {
    fun sha256(data: ByteArray): ByteArray
}

// In androidMain
actual object CryptoUtils {
    actual fun sha256(data: ByteArray): ByteArray {
        val digest = MessageDigest.getInstance("SHA-256")
        return digest.digest(data)
    }
}
```

**2. Use KMP-Compatible Libraries:**
- ‚úÖ `kotlinx.coroutines` - Cross-platform concurrency
- ‚úÖ `kotlinx.serialization` - Cross-platform serialization
- ‚úÖ Kotlin stdlib - Standard collections

**3. Avoid Platform-Specific Imports:**
- ‚ùå `android.util.Log`
- ‚ùå `java.security.*`
- ‚ùå `java.util.concurrent.*`
- ‚ùå `android.content.*`

**4. Use Proper Kotlin Syntax:**
- ‚úÖ `mutableMapOf<K, V>()` - Function call
- ‚ùå `MutableMap<K, V>()` - Interface, can't instantiate
